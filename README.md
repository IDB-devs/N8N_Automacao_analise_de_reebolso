# N8N_Automacao_analise_de_reebolso
Criar um agente de IA para análise de reembolsos. Utiliza um Webhook para receber dados de um formulário, que são combinados com informações de uma planilha. Esses dados são organizados em um JSON. Por fim, aplicamos condições que avaliam o tempo da solicitação, o nível de satisfação e o valor gasto para definir o tratamento do pedido de reembolso.
